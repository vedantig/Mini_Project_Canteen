FAQ

1)Where to go if I have complaint regarding something?

You can meet the canteen manager and try to resolve the issue. If not satisfied, you can even write a complaint letter to the college dean.

2)What if I am not able to pay money on time?

This is a canteen policy , to be able to deliver food only when paid. But you can talk to the canteen manager for some discount.

3)Is there any possibility for non-vegetarian delicacies?

We apologize. This is a Jain college, so we try to deliver 100% pure vegetarian food only.

4)Are there any possibilities of increasing the variety of food?

Yes, we are gradually planning to include more dishes both healthy and tasty for our students .

5) Where to post if i have some suggestions regarding something?

You can go and talk to the canteen manager or write a letter to dean .Your intake would be highly appreciated.
